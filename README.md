# rag_system_offline

Meu primeiro projeto no GitHub 🚀  
Este repositório representa um sistema **RAG (Retrieval-Augmented Generation)** em ambiente offline, desenvolvido com foco em aprendizado e experimentação com tecnologias de IA.

---

## 📌 Visão Geral

O projeto consiste em:
- **Frontend** desenvolvido em **React**, baseado no Create React App
- **Backend** construído utilizando **Langflow**, para orquestração do fluxo RAG
- Integração entre interface e pipeline de recuperação e geração de respostas

O objetivo principal é estudar e aplicar conceitos de **IA generativa**, **LLMs** e **arquitetura RAG**.

---

## 🖥️ Frontend

O frontend foi desenvolvido em React, com base no tutorial abaixo:

🔗 **Referência (YouTube)**  
[Building a RAG System – Frontend](https://youtu.be/domt_Sx-wTY?si=NhglrHK-4RMgX-R_)

Este projeto foi inicializado com [Create React App](https://github.com/facebook/create-react-app).

### Scripts disponíveis

No diretório do projeto, você pode executar:

#### `npm start`
Executa o app em modo de desenvolvimento.  
Abra [http://localhost:3000](http://localhost:3000) no navegador.

#### `npm test`
Executa os testes em modo interativo.

#### `npm run build`
Gera a versão de produção na pasta `build`.

---

## ⚙️ Backend

O backend foi desenvolvido utilizando **Langflow**, responsável por:
- Configuração do fluxo RAG
- Integração com modelos de linguagem
- Processamento e recuperação de contexto para geração de respostas

🔗 **Ferramenta utilizada:**  
O langflow foi baixado via terminal, de forma local
